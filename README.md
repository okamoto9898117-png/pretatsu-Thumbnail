# pretatsu-Thumbnail

> フォームに記入して Claudeに貼るだけで、世界水準のサムネイルが自動生成されます。

---

## 使い方

1. `thumbnail_form.txt` をダウンロード
2. 〇〇の部分を記入
3. Claudeに「このフォームからサムネイルを作成してください」と貼り付けるだけ
4. PNGが自動生成・保存されます

---

## 対応テーマ

| テーマ | 配色 |
|--------|------|
| Editorial（推奨） | 黒×白 タイポグラフィ |
| プレミアム | ネイビー×ゴールド |
| スタートアップ | 深紫×シアン |
| テクノロジー | 黒×ティール |
| 信頼・誠実 | 深緑×ゴールド |
| エネルギー | ネイビー×レッド |
| カスタム | 自由指定 |

## 対応サイズ

| サイズ | 解像度 | 用途 |
|--------|--------|------|
| 16:9 | 1920×1080 | プレゼン・一般用（推奨） |
| youtube | 1280×720 | YouTube サムネイル |
| ogp | 1200×630 | ブログ OGP |
| 1:1 | 1080×1080 | SNS 正方形 |
| 9:16 | 1080×1920 | ショート動画・リール |

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `thumbnail_form.txt` | 入力フォーム（社員記入用） |
| `create_thumbnail.js` | PNG自動生成スクリプト（Node.js） |
