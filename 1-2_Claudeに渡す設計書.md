# Claude サムネイル自動生成 設計書 v1.0

> このファイルはClaudeへの指示書です。フォームと一緒に貼り付けるだけで、誰でも世界水準のサムネイルが自動生成されます。

---

## 🎯 Pretatsuが目指すサムネイルの定義

> **「現代日本のハイセンスなビジネス媒体や広告ポスター、おしゃれな日本の雑誌のように、知的でクリーン、でも感性に刺さる編集デザイン」**

### 目指すべき3つの状態
1. **一瞬で内容が伝わる** — 見た瞬間に何のサムネイルかわかる
2. **思わずクリックしたくなる** — 続きが気になるビジュアル
3. **ブランドとして統一感がある** — Pretatsuらしさが出ている

---

## 🚫 絶対にやってはいけないこと（禁止ルール）

| 禁止事項 | 理由 |
|---------|------|
| 文字を詰め込みすぎる | 小さい画面で読めない。メインタイトルのみで勝負 |
| 同じレイアウトを毎回使う | 単調になる。構図にバリエーションを持たせる |
| 背景が白・薄いだけ | 存在感がない。コントラストを強くする |
| フォントをバラバラに使う | 統一感が失われる。テーマごとのフォントを守る |
| 色を4色以上使う | 散漫になる。3色以内に収める |
| 低解像度で出力する | ぼやける。最低1280×720px以上で出力 |

---

## ✅ 必ず守るべきデザインルール（Pretatsu標準）

### テキスト
- **メインタイトルは巨大に** — 画面の30〜50%を占めるサイズ
- **文字数は最小限** — 20文字以内を目標
- **体言止め** — 「〜する」ではなく「〜の実現」
- **コントラスト最優先** — 背景と文字の色差を最大にする

### ビジュアル
- **背景はダーク推奨** — 黒・ネイビー・深紫が最も高級に見える
- **余白を贅沢に使う** — 詰め込まない・空白＝高級感
- **アクセントカラーは1色のみ** — 強調箇所だけに使う
- **デコレーション要素は最小限** — ライン・円・グラデーションのみ

### サイズ・解像度
- **16:9** → 1920×1080px（プレゼン・一般用）
- **YouTube** → 1280×720px
- **OGP** → 1200×630px
- **1:1** → 1080×1080px（Instagram）
- **9:16** → 1080×1920px（ショート動画）

## ⚡ Claudeへの実行指示

フォームの内容を読み取り、以下を実行すること：

1. **フォームを解析** → メインタイトル・サブタイトル・テーマ・サイズを確認
2. **テーマのカラーコードを適用** → 上記テーマ仕様に従う
3. **以下のJSONを作成して `/tmp/thumbnail_input.json` に保存する**

```json
{
  "mainTitle": "（メインタイトルの値）",
  "subTitle": "（サブタイトルの値、省略時は空文字）",
  "category": "（カテゴリの値、省略時は空文字）",
  "brand": "（ブランド名の値、省略時は空文字）",
  "theme": "（テーマの値）",
  "aspectRatio": "（アスペクト比の値）",
  "bgColor": "（カスタム背景カラー、カスタム時のみ）",
  "accentColor": "（カスタムアクセントカラー、カスタム時のみ）",
  "textColor": "（カスタムテキストカラー、カスタム時のみ）"
}
```

4. **以下のコマンドを実行する**
```bash
node /tmp/pptx_build/create_thumbnail.js /tmp/thumbnail_input.json [出力パス]
```
- 出力パスはフォームの「ファイル名」と「保存先」から決定
- 省略時は `/Users/koudai/My Campany/thumbnail.png`

5. **完了後** → 生成されたファイルパスをユーザーに伝える

### アスペクト比対応表
| フォームの記載 | JSONの値 |
|---|---|
| 16:9（YouTube・プレゼン・一般用） | `16:9` |
| 1:1（SNS正方形） | `1:1` |
| youtube（YouTube専用） | `youtube` |
| ogp（ブログOGP） | `ogp` |
| 9:16（縦型） | `9:16` |
